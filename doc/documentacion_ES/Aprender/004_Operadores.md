# Operadores

## Matemáticas
Operadores para hacer matemáticas con números (int y float)
### Sumar
Operador: +
Ejemplo de uso:
``` Infernal
x = 6 + 2 #guarda el valor de 8
x = x + 2 #guarda el valor de 10
```
### Restar
Aquí basicamente tienes que recordar lo que diste en primaria de números negativos (menos + menos = más, más + más = más, menos + más = menos y más + menos = menos).
Operador: -
Ejemplo de uso:
``` Infernal
a = 6 - 16   #guarda el valor de -10
z = 16 - 6   #guarda el valor de 10
y = -5 -6    #guarda el valor de -11
x = -5 - -6  #guarda el valor de 1
```
### Multiplicar
Es multiplicar normal, solo que el operador no es x o · como en primaria y ESO.
Operador: *
